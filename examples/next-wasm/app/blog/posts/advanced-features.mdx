# Advanced Features

## Frontmatter Parsing

<Callout type="success">
bunny-next automatically parses YAML frontmatter!
</Callout>

```typescript
const mdxSource = await serialize(content, {
  parseFrontmatter: true,
});
console.log(mdxSource.frontmatter);
```

## Scope Variables

Pass dynamic data to your MDX:

```typescript
const mdxSource = await serialize(content, {
  scope: { userName: 'Alice', count: 42 }
});
```

## Cache Management

```typescript
import { configureMDXCache } from 'bunny-next';

configureMDXCache({
  maxSize: 1000,
  ttl: 1000 * 60 * 30 // 30 minutes
});
```
